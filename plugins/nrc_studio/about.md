NRC Studio is the official Blockbench plugin for creating content for NoRiskClient.

## Features

- **Emote Creation** — Start from the default NRC player skeleton and animate emotes
- **Native Bezier Export** — Exports bezier keyframes with `lerp_mode: "bezier"` and handle data instead of baking to linear
- **Model & Animation Export** — Export bedrock `.geo.json` models and `.animation.json` files ready for NRC

## Usage

1. File > New > **NRC Studio Model**
2. Select model type (e.g. Emote)
3. Animate using the built-in Blockbench animation tools
4. Export via File > Export > **Export NRC Animation**
